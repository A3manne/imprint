Welcome to Joomla-Imprint.

This is a Joomla! CMS component, which displays and manage Imprints.

System requirements

For Imprint Version 3

  • Joomla! CMS 1.7.x or 2.5.x

For Imprint Version 4 (not yet released)

  • Joomla! CMS 3.0.x and above

Special notice for Update prior Imprint 3.1.0

You have to reinstall this component. Normaly we do this for you with a migration script. If this migration fails you must uninstall the old version (com_impressum) manually.
